<h3 align="center">screamd</h3>
<p align="center">a simple daemon that monitors system memory and plays a sound when available memory falls below a configured threshold</p>

### Prerequisite

- [gcc](https://gcc.gnu.org/)
- [alsa](https://www.alsa-project.org/wiki/Main_Page)

### Usage

Start daemon using make.

```sh
make startd
```
